# BioFoundation

An ecosystem for biological foundation models.

## Installation

Create and activate a virtual environment:

```bash
uv venv
source .venv/bin/activate
```

Install the package:

```bash
uv pip install -e .
```

## Additional Dependencies

```bash
# PlantCad1
uv pip install -e .[mamba]
```

## Usage

Run the example script:

```bash
source .venv/bin/activate
python example.py
```
